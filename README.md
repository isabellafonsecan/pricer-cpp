# Projet de Pricer d'options ENSAE Paris

## Compilation du code
Sur la ligne de commande, dans le dossier où se trouve main.cpp exécuter les commandes suivante:
g++ ../main.cpp -o main
./main

## Code des options
On utilisera le code suivant pour les options. Le code n'est pour le moment utile que pour le black,scholes pricer 

- Vanilla European Call: VANEUCA
- Vanilla European Put: VANEUPUT
- Vanilla American Call: VANAMCA
- Vanilla American Put: VANAMPU
- Exotic Asian Call: EXASICA
- Exotic Asian Put: EXASIPUT
- Exotic Cliquet Option Call: EXCLICA
- Exotic Cliquet Option Put: EXCLIPUT
- Exotic Lookback Option Call: EXLOOCA
- Exotic Lookback Option Put: EXLOOPUT
- Exotic Option A Barrière Desactivante Call: EXBKOCA
- Exotic Option A Barrière Desactivante Put: EXBKOPUT
